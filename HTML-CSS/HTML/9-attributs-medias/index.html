Les attribts et les medias

Les attributs

Un attribut est un élément qui est placé à l'intérieur d'une balise.
Un attribut est composé de 2 parties: la clé et la valeur

Les class

les attributs de class permettent de classer les éléments afin de mieux les identifiés . On utilise très souvent les attributs de class dans notre feuille de style et dans notre script
Sa synthaxe est : `class="valeur"`

On peut avoir plusieurs class dans un même attribut et plusieurs balise peuvent avoir les mêmes class

Les ID

Un id permet de créer une référence unique à un élément. Il ne peut-être dupliqué dans un document html.
sa synthaxe : `id=valeur`

Les liens hypertextes

Les liens permettent de faire la maillage interne et externe de notre page.C'est-à-dire qu'ils permettent de naviguer dans notre site web

Les liens externes

Comme son nom l'indique , un lien externe permet de naviguer vers un autre site web noreferer: indique au navigateur, lorsque celui-ci ouvre la ressource cible, d'omettre l'en-tête Referer et de ne divulguer aucune information de referrer noopener:indique au navigateur d'ouvrir la ressource cible sans que le nouveau contexte de navigation ne reçoive d'accès au document précédent rel: précise la relation entre la source et la destination target : Précise si on doit ouvrir le lien dans un nouvel onglet(_blank) ou pas

Les videos

La balise video permet d'intégrer une video dans notre page web.
les attribut :
controls : affiche le panneau de contrôle de la video(play/pause etc..)
autoplay: lance automatiquement la video au chargement de la page
loop : permet de lire la video en boucle
muted: pour couper le son

L'avantage d'utiliser cette synthaxe pour les vidéos est qu'on peut inclure pluisieurs source de video ou pluisieurs format et aussi d'afficher un message en cas d'incapacité du navigateur à lire les formats qui sont dans la source

Ajout de CSS

Nous avons 3 possibilités d'ajouter du css dans notre code:dans le head, dans la balise en utilisant l'attribut style et dans une feuille de style séparée(ce qui est recommandé)

Dans le head

Pour ajouter du style à notre code dans le head, il faut ajouter une balise `style` et à l'intérieur écrire notre css

Ajout de JavaScript

Nous avons 2 possibilités pour ajouter du JS dans notre code

Dans une balise script

Nous pouvons écrire notre code javascript dans une balise script à la fin de notre document , juste avant la fermeture du body

Dans un fichier séparé.js

Nous pouvons et il est récommandé d'écrire notre script dans un fichier séparé. Afin de lier ce fichier à notre document, nous allons utiliser la balise script avec l'attribut src qui va préciser le chemin vers le fichier js. Cette balise est placée en bas du document avant la fermeture du body

Les listes

Les listes sont des éléments qui permettent de structurer du contenu . Il existe trois types de listes : Les listes ordonnées, les listes non ordonnées et les listes définies.

Les listes ordonnées

  1. HTML
  2. CSS
  3. JavaScript
  4. React
  5. Node.js

Les listes non ordonnées

  • France
  • Italie
  • Espagne
  • Allemagne
